Given an unsorted array of integers `nums`, return _the length of the longest consecutive elements sequence._

You must write an algorithm that runs in `O(n)` time.

**Example 1:**

**Input:** nums = [100,4,200,1,3,2]
**Output:** 4
**Explanation:** The longest consecutive elements sequence is `[1, 2, 3, 4]`. Therefore its length is 4.

**Example 2:**

**Input:** nums = [0,3,7,2,5,8,4,6,0,1]
**Output:** 9

**Constraints:**

- `0 <= nums.length <= 105`
- `-109 <= nums[i] <= 109`



One way to solve this would be to first sort the array. Then we could easily traverse the array in order to find the length of the longest consecutive sequence. The problem with this approach, though, is that the cost of sorting is O(n log n). So in order to write an algorithm that runs in O(n) time, we need to figure out how to look up each element in the array in O(1) time.

We can do this by converting the array of integers into a set of integers. Since sets use a hash function to generate a hash code for each value, we can look up each element in O(1) time (or more precisely, the amortized complexity of set lookup is O(1)).

We'll then traverse the array using `start` and do the following:

1. If `start - 1` is in the set, then that means `start` is NOT the start of a consecutive sequnce, since we could have started at `start - 1` instead. So we'll just move on to the next integer
2. If `start - 1` is NOT in the set, then that means it IS the start of a consecutive sequence. We'll initialize a variable `end` and set it equal to `start + 1` and see how long the sequence goes
3. While `end` is in `num_set`, we'll continue to increment `end` to see how long the sequence can go. Once `end` is no longer in `num_set`, the sequence has ended and we'll calculate the length.
4. We'll do this by calculating `end - start`, and update `length` to be the length of the longest consecutive sequence that we've encountered so far.

# Code

```python
class Solution:
    def longestConsecutive(self, nums: List[int]) -> int:
        num_set = set(nums)
        length = 0
        for start in nums:
            if start - 1 not in num_set:
                end = start + 1
                while end in num_set:
                    end += 1
                length = max(length, end - start)
        return length
```
