Given a string `s`, find the length of the **longest** **substring** without duplicate characters.

**Example 1:**

**Input:** s = "abcabcbb"
**Output:** 3
**Explanation:** The answer is "abc", with the length of 3. Note that `"bca"` and `"cab"` are also correct answers.

**Example 2:**

**Input:** s = "bbbbb"
**Output:** 1
**Explanation:** The answer is "b", with the length of 1.

**Example 3:**

**Input:** s = "pwwkew"
**Output:** 3
**Explanation:** The answer is "wke", with the length of 3.
Notice that the answer must be a substring, "pwke" is a subsequence and not a substring.

**Constraints:**

- `0 <= s.length <= 5 * 104`
- `s` consists of English letters, digits, symbols and spaces.

# Sliding window (w/o HashTable)
```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        seen = set()
        left = 0
        result = 0

        # The right pointer marches forward automatically
        for right in range(len(s)):
            
            # If the current letter is already in our window, our window is INVALID!
            # Keep shrinking the left side until the duplicate is kicked out.
            while s[right] in seen:
                seen.remove(s[left])
                left += 1
            
            # Now that the window is clean and valid, add the new letter
            seen.add(s[right])
            
            # Check if this new valid window is our longest one yet
            # (right - left + 1 gives us the exact length of the window)
            result = max(result, right - left + 1)

        return result
```

# Using HashTable
```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        # Dictionary to store {character: most_recent_index}
        char_index_map = {}
        left = 0
        max_length = 0

        for right in range(len(s)):
            current_char = s[right]
            
            # If we've seen this character before, AND it's inside our current window
            if current_char in char_index_map and char_index_map[current_char] >= left:
                # TELEPORT the left pointer instantly past the old duplicate
                left = char_index_map[current_char] + 1
            
            # Record/Update the exact index where we just saw this character
            char_index_map[current_char] = right
            
            # Calculate the window size
            max_length = max(max_length, right - left + 1)

        return max_length
```
