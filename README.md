## 🧠 Approach & Methodology

For every challenge, I follow a rigorous documentation process:

1. **Intuition:** Identifying the pattern (e.g., "This is a Greedy problem").
    
2. **Complexity Analysis:**
    
    - **Time Complexity:** Expressed in Big O notation, such as $O(n \log n)$ or $O(n^2)$.
        
    - **Space Complexity:** Evaluation of auxiliary data structures like $O(n)$ for Hash Maps.
        
3. **Optimization:** Iterating from a brute-force approach to the most efficient solution.
