# Approach

This is based on Python. Other might be differnt a bit

1. Initialize Variables and Data Structures:
    
    - Create a class namedô `Solution`.
    - Define a method within the class calledô `countBits`, which takes an integerô `n`ô as an argument.
    - Create an arrayô `dp`ô of lengthô `(n + 1)`ô to store the count of set bits for numbers from 0 toô `n`.
    - Initialize a variableô `sub`ô with the value 1.
2. Iterate through Numbers:
    
    - Use aô `for`ô loop to iterate over numbers from 1 toô `n`, inclusive.
3. Determine Subsequence Reset:
    
    - Check ifô `sub`ô multiplied by 2 is equal to the current value ofô `i`.
    - If true, update the value ofô `sub`ô to be equal toô `i`. This step is responsible for resetting the subsequence when the current number is a power of 2.
4. Count Set Bits:
    
    - Calculate the count of set bits for the current number by accessingô `dp[i - sub]`ô and adding 1.
    - This step utilizes the fact that the count of set bits for a numberô `i`ô can be obtained by counting the set bits of the numberô `i - sub`ô and adding 1.
5. Store Result in dp Array:
    
    - Assign the calculated count of set bits for the current number toô `dp[i]`.
6. Return Result:
    
    - After the loop completes, theô `dp`ô array will contain the count of set bits for numbers from 0 toô `n`.
    - Return theô `dp`ô array as the final result of theô `countBits`ô method.

In summary, the code implements a dynamic programming approach to count the number of set bits (1s) in the binary representation of numbers from 0 toô `n`. It uses an arrayô `dp`ô to store the results and calculates the count of set bits step by step, based on the idea of resetting subsequences at powers of 2.

class Solution:
    def countBits(self, n: int) -> List[int]:
        dp = [0] * (n + 1)
        sub = 1

        for i in range(1, n + 1):
            if sub * 2 == i:
                sub = i
            
            dp[i] = dp[i - sub] + 1
        
        return dp
