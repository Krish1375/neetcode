Given an array of stringsô `strs`, groupô **the anagrams**ô together. You can return the answer inô **any order**.

Anô **Anagram**ô is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

**Example 1:**

**Input:** strs = "eat","tea","tan","ate","nat","bat"
**Output:** "bat","nat","tan","ate","eat","tea"

**Example 2:**

**Input:** strs = ""
**Output:** ""

**Example 3:**

**Input:** strs = "a"
**Output:** "a"

**Constraints:**

- `1 <= strs.length <= 104`
- `0 <= strs[i].length <= 100`
- `strs[i]`ô consists of lowercase English letters.



# Intuition:

The intuition is to group words that are anagrams of each other together. Anagrams are words that have theô `same`ô characters but in aô `different`ô order.

# Explanation:

Let's go through the code step by step using the example inputô `["eat","tea","tan","ate","nat","bat"]`ô to understand how it works.

1. **Initializing Variables**
    
    - We start by initializing an empty unordered map calledô `mp`ô (short for map), which will store the groups of anagrams.
2. **Grouping Anagrams**  
    We iterate through each word in the input vectorô `strs`. Let's take the first word, "eat", as an example.
    
    - **Sorting the Word**  
        We create a string variable calledô `word`ô and assign it the value of the current word ("eat" in this case).
        
        Next, we sort the characters inô `word`ô using theô `sort()`ô function. After sorting,ô `word`ô becomes "aet".
        
    - **Grouping the Anagram**  
        We insertô `word`ô as the key into theô `mp`ô unordered map usingô `mp[word]`, and we push the original word ("eat") into the vector associated with that key usingô `mp[word].push_back(x)`, whereô `x`ô is the current word.
        
        Since "aet" is a unique sorted representation of all the anagrams, it serves as the key in theô `mp`ô map, and the associated vector holds all the anagrams.
        

For the given example, theô `mp`ô map would look like this after processing all the words:

```json
{
  "aet": ["eat", "tea", "ate"],
  "ant": ["tan", "nat"],
  "abt": ["bat"]
}
```

3. **Creating the Result**  
    We initialize an empty vector calledô `ans`ô (short for answer) to store the final result.
    
    - We iterate through each key-value pair in theô `mp`ô map using a range-based for loop. For each pair, we push the vector of anagrams (`x.second`) into theô `ans`ô vector.

For the given example, theô `ans`ô vector would look like this:

```javascript
[
  ["eat", "tea", "ate"],
  ["tan", "nat"],
  ["bat"]
]
```

4. **Returning the Result**  
    We return theô `ans`ô vector, which contains the groups of anagrams.

```python
class Solution:
    def groupAnagrams(self, strs):
        anagram_map = defaultdict(list)
        
        for word in strs:
            sorted_word = ''.join(sorted(word))
            anagram_map[sorted_word].append(word)
        
        return list(anagram_map.values())
```

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
```

# Step by Step Algorithm

### 1.ô **Initialize a Dictionary to Store Grouped Anagrams**

```python
ans = defaultdict(list)
```

- **Explanation**:
    - Aô `defaultdict`ô is created withô `list`ô as the default factory. This means that if a key does not exist in the dictionary, it automatically creates an empty list for that key.
    - This dictionary (`ans`) will be used to store lists of anagrams grouped by a common key.

### 2.ô **Iterate Over Each String in the Input List**

```python
for s in strs:
```

- **Explanation**:
    - A loop is initiated to iterate over each string (`s`) in the input listô `strs`.
    - The goal of this loop is to group each string with its anagrams.

### 3.ô **Sort Each String to Create a Key**

```python
key = "".join(sorted(s))
```

- **Explanation**:
    - The stringô `s`ô is sorted alphabetically using theô `sorted()`ô function. Sorting rearranges the characters of the string in a defined order (lexicographical order).
    - The sorted characters are then joined back together to form a new string (`key`).
    - This sorted string (`key`) acts as a unique identifier for anagrams. For example, bothô `"eat"`ô andô `"tea"`ô will produce the keyô `"aet"`.

### 4.ô **Group the String Under Its Anagram Key**

```python
ans[key].append(s)
```

- **Explanation**:
    - The stringô `s`ô is appended to the list in the dictionaryô `ans`ô corresponding to its key (`key`).
    - If the key does not already exist in the dictionary,ô `defaultdict`ô automatically creates it with an empty list, so no need to check if the key exists.
    - This groups all anagrams together in the same list.

### 5.ô **Return the Grouped Anagrams**

```python
return ans.values()
```

- **Explanation**:
    - The method returns the values of the dictionaryô `ans`.
    - Since the dictionary's values are lists of grouped anagrams, this effectively returns a list of lists, where each inner list contains strings that are anagrams of each other.

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:    
        ans = defaultdict(list)

        for s in strs:
            count = [0] * 26

            for c in s:
                count[ord(c) - ord("a")] += 1
            ans[tuple(count)].append(s)
        
        return list(ans.values())
```

# Step by Step Algorithm

### 1.ô **Initialize a Dictionary to Store Grouped Anagrams**

```python
ans = collections.defaultdict(list)
```

- **Explanation**:
    - Aô `defaultdict`ô of lists is created. This dictionary will be used to store lists of strings that are anagrams of each other.
    - The keys in this dictionary will be tuples representing the frequency of each character in a string.

### 2.ô **Iterate Over Each String in the Input List**

```python
for s in strs:
```

- **Explanation**:
    - The loop starts to iterate over each string (`s`) in the input listô `strs`.
    - The goal is to group each string with its corresponding anagrams.

### 3.ô **Initialize a Character Count Array**

```python
count = [0] * 26
```

- **Explanation**:
    - A listô `count`ô of size 26 is initialized with zeros. This list will keep track of the frequency of each letter in the current stringô `s`.
    - Each index of the list corresponds to a letter in the alphabet (`'a'`ô toô `'z'`).

### 4.ô **Count the Frequency of Each Character in the String**

```python
for c in s:
    count[ord(c) - ord("a")] += 1
```

- **Explanation**:
    - Another loop iterates over each characterô `c`ô in the stringô `s`.
    - `ord(c) - ord("a")`ô calculates the index in theô `count`ô list that corresponds to the characterô `c`ô by finding the difference between the ASCII values ofô `c`ô andô `'a'`.
    - The value at this index is incremented by 1, effectively counting the occurrence of each character in the string.

### 5.ô **Convert the Character Count Array to a Tuple and Use It as a Key**

```python
ans[tuple(count)].append(s)
```

- **Explanation**:
    - Theô `count`ô list is converted into a tuple because tuples are hashable and can be used as keys in a dictionary.
    - The stringô `s`ô is then appended to the list associated with this tuple key in theô `ans`ô dictionary.
    - This step groups all strings that have the same character frequency under the same key, which means they are anagrams of each other.

### 6.ô **Return the Grouped Anagrams**

```python
return ans.values()
```

- **Explanation**:
    - The method returns the values of theô `ans`ô dictionary, which are lists of grouped anagrams.
    - This effectively returns a list of lists, where each inner list contains strings that are anagrams of each other.

### Summary

- The algorithm counts the frequency of each character in each string and uses this frequency count (in the form of a tuple) as a key to group anagrams together.
- Anagrams will have identical character frequency distributions and thus be grouped under the same key in the dictionary. The final result is a collection of these grouped anagrams.
