Given the roots of two binary trees `p` and `q`, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical, and the nodes have the same value.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/12/20/ex1.jpg)

**Input:** p = [1,2,3], q = [1,2,3]
**Output:** true

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/12/20/ex2.jpg)

**Input:** p = [1,2], q = [1,null,2]
**Output:** false

**Example 3:**

![](https://assets.leetcode.com/uploads/2020/12/20/ex3.jpg)

**Input:** p = [1,2,1], q = [1,1,2]
**Output:** false

**Constraints:**

- The number of nodes in both trees is in the range `[0, 100]`.
- `-104 <= Node.val <= 104`

# DFS
```python
class Solution:
    def isSameTree(self, p: Optional[TreeNode], q: Optional[TreeNode]) -> bool:
        # 1. Both hit a dead end: Perfect match!
        if not p and not q:
            return True
        
        # 2. One hit a dead end, or values don't match: Mismatch!
        if not (p and q) or p.val != q.val:
            return False
            
        # 3. Current nodes match! Send the scouts down both sides.
        return self.isSameTree(p.left, q.left) and self.isSameTree(p.right, q.right)
```

# BFS
```python
from collections import deque

class Solution:
    def isSameTree(self, p: Optional[TreeNode], q: Optional[TreeNode]) -> bool:
        # Initialize the queue with our first pair (the roots)
        queue = deque([(p, q)])
        
        while queue:
            # Pop the pair
            node_p, node_q = queue.popleft()
            
            # 1. Double Dead End: Perfect match for this path!
            # We use 'continue' to skip adding children, but we don't return True yet
            # because there might be other nodes waiting in the queue.
            if not node_p and not node_q:
                continue
                
            # 2 & 3. Mismatched Dead End OR Value Mismatch
            if not node_p or not node_q or node_p.val != node_q.val:
                return False
                
            # 4. We survived the checks! The nodes match. 
            # Pair up their left children and right children for the next level.
            queue.append((node_p.left, node_q.left))
            queue.append((node_p.right, node_q.right))
            
        # If the queue empties completely and we never hit a False, the trees are identical!
        return True
```
