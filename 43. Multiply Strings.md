Given two non-negative integers `num1` and `num2` represented as strings, return the product of `num1` and `num2`, also represented as a string.

**Note:** You must not use any built-in BigInteger library or convert the inputs to integer directly.

**Example 1:**

**Input:** num1 = "2", num2 = "3"
**Output:** "6"

**Example 2:**

**Input:** num1 = "123", num2 = "456"
**Output:** "56088"

**Constraints:**

- `1 <= num1.length, num2.length <= 200`
- `num1` and `num2` consist of digits only.
- Both `num1` and `num2` do not contain any leading zero, except the number `0` itself.



```python
class Solution:
    def multiply(self, num1: str, num2: str) -> str:
        # Create our grade-school worksheet
        result = [0] * (len(num1) + len(num2))

        # Iterate backwards through both strings
        for i in range(len(num1) - 1, -1, -1):
            for j in range(len(num2) - 1, -1, -1):
                
                # 1. Calculate the product
                product = int(num1[i]) * int(num2[j])
                
                # 2. Add whatever is already sitting in our target "ones" place
                total = product + result[i + j + 1]
                
                # 3. Overwrite the "ones" place with the new digit
                result[i + j + 1] = total % 10
                
                # 4. Add the carry to the "tens" place
                result[i + j] += total // 10

        # Convert the integer array to a single string
        res_str = "".join(str(digit) for digit in result)
        
        # Strip leading zeros, but default to "0" if the whole string is empty
        return res_str.lstrip('0') or "0"
```
