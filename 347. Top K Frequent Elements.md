# [[- Arrays and Hashing]]
# [347. Top K Frequent Elements](https://leetcode.com/problems/top-k-frequent-elements/)

#array #medium 
[[- Medium Level]]
Given an integer array `nums` and an integer `k`, return _the_ `k` _most frequent elements_. You may return the answer in **any order**.

**Example 1:**

**Input:** nums = [1,1,1,2,2,3], k = 2
**Output:** [1,2]

**Example 2:**

**Input:** nums = [1], k = 1
**Output:** [1]

**Constraints:**

- `1 <= nums.length <= 105`
- `-104 <= nums[i] <= 104`
- `k` is in the range `[1, the number of unique elements in the array]`.
- It is **guaranteed** that the answer is **unique**.


My Code:

```python
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:
        hashmap = defaultdict(list)
        for num in nums:
            hashmap[num] = hashmap.get(num, 0) + 1

        sort = {k: v for k, v in sorted(hashmap.items(), key=lambda item: item[1])}

        ans = []
        for i in range(1,k+1):
            ans.append(list(sort.keys())[-i])
        return ans
```

# Intuition

A Python dictionary can be used to store unique values as well as the frequency of their occurence. Once obtained, we return the top k most frequently occuring elements.

# Approach

- Count frequencies using a dictionary.
- Sort the elements by frequency in descending order.
- Return the first k elements from this sorted list.

# Complexity

- Time complexity: The costliest operation here is sorting, which operates in O(nlogn) time complexity. Note that this solution can be further improved to O(nlogk) complexity by using max-heaps to store the key of the most frequently occuring values.

- Space complexity: Creating an additional dictionary makes the space complexity O(n).

# Code

```java
class Solution:
    def topKFrequent(self, nums: List[int], k: int) -> List[int]:

        orb = {}

        for i in nums:
            if i in orb:
                orb[i] += 1
            else: orb[i] = 1
        
        sorted_orb = sorted(orb, key=orb.get, reverse=True)
        return sorted_orb[:k]
                    
```
